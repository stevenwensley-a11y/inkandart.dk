SELF-HOSTED FONTS — LICENSE + PROVENANCE

These fonts are bundled with the site to avoid third-party requests to Google
Fonts (privacy + perf). They are licensed under the SIL Open Font License 1.1,
which permits embedding and redistribution.

## Files

| File | Family | Weight | Subset | Source |
|---|---|---|---|---|
| `BebasNeue-Regular.woff2` | Bebas Neue | 400 | Latin (U+0000–00FF + common typography) | Google Fonts CDN, v16 |
| `SpaceMono-Regular.woff2` | Space Mono | 400 | Latin | Google Fonts CDN, v17 |
| `SpaceMono-Bold.woff2` | Space Mono | 700 | Latin | Google Fonts CDN, v17 |

Latin subset covers all Danish characters (æ, ø, å, Æ, Ø, Å are all in the
Latin-1 supplement at U+00E6 / U+00F8 / U+00E5 etc.). If we later need Latin
Extended (accents on imported names from outside DA/EN), download the
`latin-ext` subsets from the same Google Fonts CSS endpoint.

## How to refresh

```sh
UA="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 Chrome/121.0.0.0 Safari/537.36"
curl -sL -A "$UA" "https://fonts.googleapis.com/css2?family=Bebas+Neue&family=Space+Mono:wght@400;700&display=swap"
# Find the `latin` block for each face → grab the woff2 URL → curl -o the file.
```
